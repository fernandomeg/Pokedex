# Pokedex Challenge
##### Este proyecto tiene por objetivo mostrar una Pokédex que nos permita visualizar el listado de Pokemon's así como sus caracteristicas generales como sus Habilidades o su Línea de Evolución. También te permite añadir una marca de tus Pokemon's favoritos
